# 后现代主义个人博客

一个纯静态的个人博客网站，采用后现代主义设计风格，以白色背景和黑色线条为特色。

## 设计理念

本项目采用后现代主义设计理念，强调：
- **极简主义**：去除所有不必要的元素，只保留核心内容和功能
- **网格系统**：使用清晰的网格布局创造秩序感
- **白色空间**：大量留白提供视觉呼吸空间
- **黑白对比**：强烈的黑白对比创造视觉冲击力
- **解构主义**：打破传统布局规则，创造视觉张力

## 技术实现

- **纯静态网站**：不依赖任何静态生成框架（Hexo、Hugo、Jekyll等）
- **HTML5**：语义化标记，提高可访问性
- **CSS3**：现代CSS特性，包括Grid和Flexbox布局
- **原生JavaScript**：无依赖的交互功能
- **响应式设计**：适配各种屏幕尺寸

## 项目结构

```
pecmens.github.io/
├── assets/
```

## 功能特性

- **响应式布局**：适配桌面、平板和手机
- **平滑滚动**：页面内锚点平滑滚动
- **移动端菜单**：响应式导航菜单
- **滚动动画**：页面元素滚动显示动画
- **悬停效果**：交互元素的悬停状态
- **加载动画**：页面加载时的渐入效果

## 本地预览

1. 克隆或下载项目到本地
2. 使用任何HTTP服务器（如Python的SimpleHTTPServer、Node.js的http-server等）在项目根目录启动服务器
3. 在浏览器中访问 `http://localhost:8000`

例如，使用Python启动服务器：
```bash
python -m http.server 8000
```

或使用Node.js：
```bash
npx http-server
```

## 自定义指南

### 添加新博客文章

1. 在 `blog/` 目录下创建新的HTML文件
2. 复制现有文章的结构和样式
3. 更新主页 `index.html` 中的文章列表

### 修改样式

主要样式定义在 `assets/css/main.css` 文件中：
- 颜色变量在文件顶部定义
- 网格背景样式在 `.grid-background` 类中
- 响应式断点在媒体查询中定义

### 添加交互功能

JavaScript代码在 `assets/js/main.js` 文件中：
- 页面加载动画
- 滚动效果
- 移动端菜单切换
- 平滑滚动
